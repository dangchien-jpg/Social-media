# Social Media Application
A social media application that allows users to authenticate, connect with others, share posts, and interact through likes and comments.  
This project focuses on backend logic, RESTful APIs, and database design, combined with a ReactJS frontend.

### Authentication & User
- Register and login
- JWT-based authentication (Access Token)
- Update user profile information

### Social Features
- Follow / Unfollow users
- Create, update, delete posts
- Like / Unlike posts
- Comment on posts

### Backend Highlights
- RESTful API design
- Authentication & authorization using JWT
- Modular backend structure (routes, controllers, models)
- API testing with Postman
